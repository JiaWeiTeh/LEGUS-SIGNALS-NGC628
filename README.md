# LEGUS-SIGNALS-NGC628

<p><h2>Repo Structure</h2></p>
<ul>
    <li><code>./lib:</code> Directory containing libraries used by this paper. Note: the library will not be included here as of now; in future we will provide methods to access the libraries.
    <ul class="square">
          <li><code>/LEGUS:</code> LEGUS's processed catalogue. </li>
          <li><code>/SIGNALS:</code> SIGNALS's processed catalogue. </li>
          <li><code>/SLUG2:</code> SLUG data. </li>
        </ul></li>
    <li><code>./src:</code> Directory containing source code for paper.
        <ul class="square">
          <li><code>/tools:</code> Contains useful tools to analyse data.</li>
          <li><code>/dat:</code> Directores where all saved data (e.g., .py files) are stored.</li>
          <li><code>/plots:</code> Contains scripts which plots figures in paper.</li>
          <li><code>/figs:</code> Contains figures; this is the output from the <code>/plots</code> directory.</li>
        </ul></li>
</ul>
